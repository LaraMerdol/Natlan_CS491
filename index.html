PROJECT PROPOSAL

Introduction

The main purpose of this page is to present a description of the Natlan application.

Project Description

The purpose of this project is to neutralize gender discrimination (extandable to race) by using natural language processing. It will be applicable as chrome extension or keyboard extension (creating a new keyboard). It will process voices or text to target stereotypes and recommand or recorrect the content. Additional mobile application can be developed to analyze user input. The project can be taught as a different approach to grammarly.
